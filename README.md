# Description

There's an unused block in the game named the 'Fletching Table'. So, I decided to make this block useful by adding a much simple way to craft tipped arrows.

This mod will give the ability to the Fletching Table block, by converting a normal arrow to a tipped arrow using a potion.
One potion (any type) can transform any number of arrow from the first slot.
